# 정책자금자동화 v3.1.1 Change Log

## 변경 목적
- v3.1.0 UI 리뉴얼 이후 사이드바 로고 가독성과 메뉴 디자인을 추가 개선했습니다.
- 파란 사이드바에서 오아시스 로고가 흐릿하게 보이는 문제를 보정하고, 메뉴 아이콘 중복 느낌을 제거했습니다.

## 수정 파일
- app.py
- ui.py
- VERSION.txt
- CHANGELOG_v3.1.1.md

## 주요 변경사항
1. 사이드바 오아시스 로고 확대 및 흰색 표시 보정
2. 사이드바 브랜드 문구를 `오아시스 세무회계 / OASIS TAX & ACCOUNTING`으로 정리
3. 메뉴 텍스트 앞 이모지 아이콘 제거
4. 메뉴 앞 원형 표시를 흰색 채움 원으로 변경
5. 선택된 메뉴는 흰색 배경 + 파란색 텍스트로 강조
6. UI 전용 파일 `ui.py`를 신규 추가하여 향후 디자인 수정 유지보수성을 개선

## 기존 기능 영향 여부
- 기능 로직 변경 없음
- 회원가입, 관리자 승인, 크레탑 자동등록, 누적 고객DB, 정책자금 매칭 기능 유지
