# OASIS v9.3.1 적용방법

1. 현재 프로젝트 버전이 `v9.3.0`인지 확인합니다.
2. ZIP 내부 파일과 `payload` 폴더를 프로젝트 최상위 폴더에 덮어씁니다.
3. `RUN_UPDATE_v9.3.1.bat`를 실행합니다.
4. `UPDATE_OK`와 `VERSION=v9.3.1`이 표시되는지 확인합니다.
5. 앱을 재배포한 후 `기업컨설팅 > 직원현황 > 2026 고용지원금 자동 분석`에서 확인합니다.

업데이트 전 `employment_support_2026.py`와 `VERSION.txt`는 `backup/before_v9.3.1_날짜시간`에 자동 백업됩니다.
