# CHANGELOG v5.6.0

## 상담녹음 기반 정책자금 매칭키워드 자동강화

### 정책자금 신호 자동분석
- 녹취에서 정책자금 관련 발언 자동탐지
- 정책자금 관련성 여부와 신뢰도 표시
- 시설투자·기계구입·채용·R&D·수출·판로·운전자금 등의 신호 분석
- 녹취 근거와 추가 확인질문 표시

### 매칭키워드 자동반영
- 상담일지 저장 시 추천 키워드를 고객별 정책자금 매칭설정에 추가
- 기존 키워드·관심분야는 삭제하거나 덮어쓰지 않음
- 동일 키워드 중복 저장 방지
- AI 신뢰도 60점 이상인 경우에만 자동반영
- 사용자가 체크박스로 자동반영 여부를 선택 가능

### 관심지원분야 자동보완
- 운전자금
- 시설자금
- 기계·설비 구입
- 차량 구입
- 신규채용
- 고용유지
- 수출
- 판로
- R&D
- 창업
- 스마트공장

### 오탐 방지
- 이익소각·가지급금·가업승계·정관개정만 논의된 상담은
  정책자금 키워드로 억지 반영하지 않음
- 구체적인 투자·채용·자금수요 발언을 근거로 판단

### 데이터 보호
- 기존 고객DB 수정 없음
- 기존 정책자금 키워드 유지
- 새 추천키워드만 병합
