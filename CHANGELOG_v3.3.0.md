# 정책자금자동화 v3.3.0 Change Log

## 핵심 변경
- 고객관리 CRM에 중복 고객 관리 기능 추가
- 사업자등록번호, 법인등록번호, 업체명 기준 중복 고객 자동 탐지
- 중복 그룹별로 남길 고객 1건 선택 후 나머지 삭제 가능
- 개별 고객 삭제 기능 추가
- 삭제 전 자동 백업 파일 생성
- 고객 삭제 시 CRM 상담메모/타임라인 보조 기록도 함께 정리

## 수정 파일
- app.py
- crm.py
- VERSION.txt
- CHANGELOG_v3.3.0.md

## 기존 기능 영향
- 기존 고객DB 양식과 시트 구조는 유지
- 기존 매칭, 크레탑 자동등록, 회원관리 기능 삭제 없음
- 삭제 기능은 사용자가 확인 체크 후 버튼을 눌러야만 실행
