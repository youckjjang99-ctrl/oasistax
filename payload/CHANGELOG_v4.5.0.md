# CHANGELOG v4.5.0

## 기업관리센터 및 CRM 클라우드 자동복원

### 기업관리센터
- 고객 한 곳을 선택해 전체 정보를 한 화면에서 관리
- 기업 기본정보·재무정보
- CRM 상태·진행단계·중요도·담당자·예정일·상담메모
- 정책자금 매칭키워드·관심분야·제외키워드
- 등기정보·주가평가
- 기업 히스토리
- AI 빠른진단·대표 미팅 질문

### CRM 자동복원
- Streamlit Cloud 재배포 후 Supabase `oasis_crm` 자동 조회
- 고객 상태·다음 액션·예정일·상담메모 자동복원
- v4.4 진행단계·중요도·담당자 자동복원
- 상담 타임라인 중복 방지 복원
- 기존 CRM 저장 함수를 사용해 이전 화면과 호환

### 저장구조
- 기업관리센터에서 CRM 저장 시 로컬 + Supabase 동시 저장
- 정책자금 매칭설정도 로컬 + Supabase 동시 저장
- 기존 고객DB와 기존 메뉴는 그대로 유지

### 데이터 보호
- 기존 고객DB 삭제·초기화 없음
- Supabase CRM은 읽기 후 로컬 호환형으로 복원
- 기업관리센터는 기존 자료를 통합 조회하는 추가 화면
