# CHANGELOG v7.2.0

- 기업컨설팅 상단 통계·필터·빠른보기 영역 제거
- 기존 기업컨설팅 상세 화면과 탭 구성 유지
- `관리할 기업` 글자 확대

- 회사 선택창 오른쪽에 작은 `✕` 삭제 버튼 추가
- 삭제 클릭 시 `삭제하시겠습니까?` 확인창 표시

- 휴지통 화면에서 고객 복원 가능
- 동일 계정 새 로그인 시 기존 세션 자동 만료
- 관리자 포함 모든 승인 계정에 중복 로그인 방지 적용
- 기존 고객DB·CRM·상담일지·정관·정책자금·히스토리 유지
