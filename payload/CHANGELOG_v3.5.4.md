### 수정 파일
- `app.py`

### 변경사항
- PDF 앞 1~2페이지만 먼저 읽어 사업자등록번호 중복 여부 확인
- 중복 고객이면 전체 PDF 재무분석 중단
- 누적DB 전체 대신 사업자등록번호 열만 읽어 중복검사
- 저장 시 이미 분석한 데이터를 재사용하여 PDF 이중 분석 제거
- 저장 후 `st.rerun()` 제거
- session_state에는 작은 추출 결과만 보관
- 저장 직전 중복검사 재확인

### 기존 기능 영향
기존 크레탑 재무 추출, 고객관리, 정책자금 매칭 기능은 유지됩니다.
