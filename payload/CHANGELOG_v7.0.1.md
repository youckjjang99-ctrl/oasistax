# CHANGELOG v7.0.1

## 한글 OCR 정확도 및 영문 오인식 개선

- 페이지 방향 판정을 `kor` 한글 전용 OCR로 우선 수행
- 페이지마다 PSM 3·4·6·11 문단모드를 비교해 최적 결과 선택
- 한글 전용 결과가 낮은 페이지만 `kor+eng`로 재시도
- 한글 비율, 영문 비율, 정관 조항 인식률, 의미 없는 영문 토큰을 함께 평가
- `SPAN`, `end` 등 한글 문장에 섞인 비정상 영문 조각 자동 제거
- 정상적인 CEO, AI, M&A, R&D, CB, BW 등 약어는 보존
- 영문 비율이 과도하거나 한글 인식률이 낮은 페이지는 재검토 페이지로 분리
- 품질 미달 페이지는 깨진 문구 대신 `OCR 재검토 필요` 메시지를 전달
- 페이지별 사용 OCR 언어, PSM 모드, 한글·영문 비율, 재검토 여부 표시
- 기존 자동회전·기울기보정·정관 자동개정·PDF 버전관리 유지
- Supabase SQL 추가 실행 불필요
