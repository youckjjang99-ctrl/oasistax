# CHANGELOG v3.8.1

## 주가평가 입력 오류 및 고객DB 연동 개선

### 오류 수정
- Streamlit `text_input`에 정수 연도가 전달되어 발생한 TypeError 수정
- 연도, 순손익액, 발행주식수 입력 상태를 문자열로 강제 변환

### 고객DB 연동
- `기존 고객DB에서 재무정보 불러오기` 버튼 추가
- 기존 고객 기본정보, 총자산, 총부채, 당기순이익 자동 입력
- 과거 크레탑 분석 결과가 있으면 최근 3개년 재무정보 자동 입력

### 크레탑 재무 캐시
- 최초 크레탑 고객등록 시 주가평가용 재무정보를 별도 JSON에 저장
- 고객DB 파일 자체는 수정하지 않음
- 저장 파일: 회원별 `stock_financial_cache.json`

### 기존 데이터
- 패치 전 등록 고객은 현재 고객DB의 최신 재무정보만 불러옴
- 최근 3개년 정보가 필요하면 크레탑 PDF를 한 번 다시 분석하면 캐시에 저장됨
