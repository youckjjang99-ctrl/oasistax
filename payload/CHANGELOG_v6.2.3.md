# CHANGELOG v6.2.3

## 녹음파일 버튼 활성화 및 작업상태 자동복구

- 파일명이 표시돼도 생성 버튼이 계속 비활성화되던 문제 수정
- 업로드 객체 존재 여부가 아니라 실제 파일 바이트 크기로 완료상태 판단
- 업로드 완료 용량을 초록색 상태창으로 표시
- 이전 오류로 running 상태만 남은 경우 자동 초기화
- 사용자가 직접 누를 수 있는 `상태 초기화` 버튼 추가
- 30분 이상 남아 있는 비정상 작업 자동 정리
- 생성 중에는 버튼을 비활성화해 중복 API 호출 차단
- 기존 고객DB·CRM·상담일지 자료 변경 없음
