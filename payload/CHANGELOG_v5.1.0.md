# CHANGELOG v5.1.0

## 기업 컨설팅 디자인 개편 및 AI 리포트 통합

### 디자인 개편
- 기업명·사업자번호·대표자·업종·중요도를 보여주는 상단 기업 헤더
- CRM 상태와 상담 진행단계를 서로 다른 색상 카드로 명확히 구분
- 매출액과 당기순이익을 별도 재무 카드로 표시
- AI 진단의 강점·확인사항·우선 실행을 테마별 카드로 개선
- 대표 미팅 질문을 번호형 카드 목록으로 개선
- 탭 여백·테두리·그림자·반응형 레이아웃 개선

### 메뉴 정리
- 별도 `AI 컨설팅 리포트` 메뉴 제거
- 기존 기업 컨설팅의 `AI 진단` 탭에 전체 리포트 기능 통합
- 중복 메뉴를 줄이고 고객 중심 업무 흐름 유지

### AI 진단 통합
- 기업 기본정보·재무·등기·주가평가·매칭설정 통합분석
- 강점·확인사항·추천전략·대표 미팅 질문 표시
- 정보 완성도 표시
- AI 컨설팅 리포트 엑셀 다운로드

### 데이터 보호
- 기존 고객DB·CRM·Supabase 자료 변경 없음
- 화면 디자인과 메뉴 구조만 개선
