# CHANGELOG v4.4.0

## v4.3.2 주소 보완 + v4.4 CRM 고도화·기업 히스토리 통합

### 주소 자동 보완
- 사업장 소재지에서 시도·시군구 자동 분리
- 경기/서울/충남 등 정책자금 매칭용 표준 지역명 적용
- 기존 고객은 시도·시군구가 공란인 경우만 보완
- 주소 원문과 기존 입력값은 덮어쓰지 않음
- 누적 고객DB 다운로드 전 자동 보완
- 크레탑 신규 분석값에도 시도·시군구 자동 추가

### CRM 고도화
- 상담 진행단계
- 중요도 별점 1~5
- 담당자 지정
- 중요 고객·진행 중·계약 완료 요약카드
- 기존 고객 상태·다음 액션·예정일·상담메모 유지
- CRM 확장정보를 로컬과 Supabase CRM 데이터에 함께 저장

### 기업 히스토리
- 크레탑 분석 때마다 주요 기업정보 스냅샷 저장
- 매출액·영업이익·당기순이익·자산·부채·자본·종업원수 추적
- 이전 분석 대비 증가·감소 자동 표시
- 사업장 소재지 변경 표시
- 동일 데이터의 연속 중복 저장 방지
- 로컬 JSON과 Supabase 이중 저장

### 데이터 보호
- 기존 고객DB 삭제·초기화 없음
- 주소는 공란만 보완
- 고객 히스토리와 CRM 확장정보는 별도 파일에 저장
