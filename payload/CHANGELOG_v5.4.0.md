# CHANGELOG v5.4.0

## API 비용 및 처리시간 최적화

### 음성 녹취 캐시
- 녹음파일 SHA-256 해시로 동일 파일 자동 인식
- 같은 녹음·같은 모델·같은 노이즈 설정이면 기존 녹취 재사용
- 음성 API를 다시 호출하지 않아 비용과 시간을 절약

### 상담일지 캐시
- 같은 녹취·같은 요약 모델이면 기존 상담일지 재사용
- 반복 테스트 시 텍스트 생성 API 호출 생략
- 최대 200개 캐시만 유지해 저장공간 관리

### 저용량 음성 최적화
- API 전송 전 모노·16kHz·48kbps로 자동 압축
- 대화 인식에 필요한 음성 대역은 유지
- 업로드 용량과 전송시간 감소
- 25MB 초과 자동분할 횟수 감소

### 저비용 기본 모델
- 기본 음성인식 모델을 `gpt-4o-mini-transcribe`로 변경
- Secrets에서 `OPENAI_TRANSCRIPTION_MODEL`로 변경 가능
- 상담일지 모델은 기존 `OPENAI_SUMMARY_MODEL` 설정 유지

### 선택적 재생성
- 녹취는 유지하고 상담일지만 다시 생성 가능
- 문구나 분석만 다시 만들 때 음성 API 비용이 발생하지 않음
- 캐시 사용 여부를 화면에 표시

### 데이터 보호
- 기존 고객DB 수정 없음
- 캐시는 회원별 `consultation_ai_cache.json`에 별도 저장
- 원본 녹음파일은 저장하지 않음
