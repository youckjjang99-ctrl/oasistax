# OASIS 정책자금자동화 v6.3.0 ChangeLog

## 추가
- 기업컨설팅 > CRM에 `녹음파일 상담일지 보기` 탭 추가
- 기업별 저장 상담일지의 제목·요약·니즈·논의내용·추천솔루션·필요서류·후속업무·전체 녹취록 조회
- 상담일지 검색 기능 추가
- Supabase `oasis_consultation_journals` 테이블을 이용한 상담일지 영구 저장 및 로컬 기록 통합 조회

## 개선
- 상담일지 저장 시 정책자금 관련 키워드와 관심분야를 기존 수동 설정에 병합
- 중복 키워드 방지 및 기존 제외키워드·자금목적·투자계획 보존
- 저장 직후 화면 재조회로 정책자금 매칭설정과 매칭 결과에 최신 키워드 반영

## 데이터 보호
- 기존 고객DB, CRM, 상담일지 JSON, 매칭키워드, Supabase 고객 데이터 및 Storage 녹음파일을 삭제하거나 초기화하지 않음
- 신규 상담일지 테이블만 추가하며 기존 로컬 상담일지는 계속 유지
