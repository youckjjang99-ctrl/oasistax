# CHANGELOG v6.2.2

## 상담일지 JSON 오류 수정 및 대용량 처리속도 개선

### 치명적 오류 수정
- 정책자금 추천 JSON 예시의 중괄호가 f-string 서식으로 잘못 해석되던 문제 수정
- `Invalid format specifier ' true, "confidence"...'` 오류 제거
- 정책자금 추천키워드·관심분야·근거·추가질문 생성 정상화

### 처리순서 개선
기존:
1. 78MB 원본을 Supabase에 먼저 업로드
2. 녹취
3. 상담일지

변경:
1. 녹취
2. 상담일지 분석
3. 클라우드 보관
4. 결과 표시

- 긴 Supabase 업로드 때문에 녹취 시작이 늦어지는 현상 완화
- Storage 실패가 상담일지 생성을 막지 않음

### 빠른 클라우드 보관
- 기본값으로 모노·16kHz·48kbps m4a 압축본 저장
- Streamlit → Supabase 2차 업로드 용량과 시간 감소
- 체크 해제 시 원본 그대로 저장 가능
- 원본 크기와 저장 크기를 내부 메타정보에 기록

### 안내
- PC → Streamlit 최초 업로드 시간은 인터넷 업로드 속도의 영향을 받음
- 이번 패치는 최초 업로드 이후의 압축·Supabase 전송·처리 대기시간을 줄임

### 데이터 보호
- 기존 고객DB·CRM 변경 없음
- 기존 상담일지·정책자금 키워드 기능 유지
