# CHANGELOG v3.7.2

## 기존 고객 크레탑 재분석 갱신 및 AI 카드 디자인 개선

### 핵심 개선
- 중복 고객도 크레탑 전체 보고서를 분석합니다.
- 새 행을 만들지 않고 기존 고객의 누락정보를 갱신합니다.
- 사업장 소재지, 설립일, 종업원수, 매출액, 영업이익, 당기순이익,
  자산·부채·자본 정보를 누적DB 필수 컬럼으로 보강합니다.
- 중복 PDF 재등록 시 기존 행에 최신 재무정보를 반영합니다.
- AI 기업분석 카드를 대시보드형 디자인으로 개선합니다.
- 정보 완성도, 수익성 상태, 보완 필요 건수를 카드로 표시합니다.

### 기존 기능 영향
- 사업자등록번호 기준 중복 차단 유지
- 새 중복 고객 행은 추가되지 않음
- v3.6.1 안정 패키지 버전 유지
