# CHANGELOG v6.3.3

## 수정 사항
- 사업자번호의 하이픈, 공백 등 표기 차이를 제거해 기존 클라우드 녹음파일을 동일 기업으로 조회합니다.
- 사업자번호가 비어 있거나 과거 메타데이터가 불완전한 경우 같은 사용자 계정의 기업명으로 보조 조회합니다.
- 빠른 클라우드 보관으로 압축된 파일과 원본 파일의 SHA-256 값을 각각 저장합니다.
- 신규 업로드 전 원본 해시를 기준으로 기존 녹음파일을 먼저 찾아 중복 Storage 업로드를 줄입니다.
- 기존 상담일지의 사업자번호 표기 차이도 정규화하여 조회합니다.
- 기존 Supabase Storage 녹음파일과 메타데이터는 삭제하거나 이동하지 않습니다.

## 데이터 보호
- 기존 고객DB, CRM, 상담일지, 정책자금 키워드, AI 캐시, 사용자 계정 및 Storage 파일은 유지됩니다.
- Supabase 메타데이터 테이블에는 정규화 사업자번호, 원본 해시, 원본 크기 컬럼만 안전하게 추가합니다.
