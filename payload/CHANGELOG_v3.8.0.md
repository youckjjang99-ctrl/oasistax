# CHANGELOG v3.8.0

## 주가평가 모듈 추가

### 수정 파일
- `app.py`
- `cretop_worker.py`
- `VERSION.txt`

### 신규 파일
- `stock_valuation.py`

### 핵심 기능
1. 정책자금 매칭 아래에 `주가평가` 메뉴 추가
2. 기존 고객DB를 읽기 전용으로 불러오기
3. 크레탑 PDF에서 회사정보·최근 3개년 재무정보 자동 추출
4. 자동 추출이 어려운 발행주식수·세법상 조정값은 공란 유지
5. 사용자가 모든 입력값을 직접 수정 가능
6. 일반법인 3:2, 부동산과다보유법인 2:3, 순자산가치 단독 적용
7. 순자산가치 80% 하한과 순손익가치 환원율 10% 반영
8. 최대주주 할증 적용·제외를 사용자가 선택
9. 평가자료 저장·불러오기·수정 저장
10. 평가자료는 고객DB와 분리된 JSON 파일에 보관

### 데이터 보호
- 기존 고객리스트 및 누적 고객DB를 수정하지 않습니다.
- 주가평가 자료는 회원별 `stock_valuations.json`에 별도 저장합니다.
