# CHANGELOG v3.7.0

## 고객관리 CRM 고도화 및 크레탑 추출 보완

### 크레탑 추출 개선
- 설립년월, 설립년월일, 회사설립일, 법인설립일, 창업일 패턴 지원
- 주소, 사업장 소재지, 본사 소재지, 소재지 패턴 지원
- 당기순이익, 당기순손익, 순이익, 법인세차감후순이익 등 후보 계정 지원
- 요약표·상세표·문서 전체의 백만원/천원 단위 표를 순차 탐색

### CRM 기능
- 오늘 연락, 기한 경과, 7일 이내 예정 고객 집계
- 고객 상태·다음 액션·다음 예정일 관리
- 날짜 선택형 다음 예정일
- 날짜별 상담 타임라인 유지
- 고객 기본정보 직접 수정
- 규칙 기반 AI 기업분석 카드
  - 검토 포인트
  - 누락 및 확인 필요사항
  - 대표 질문

### 기존 기능 영향
- 사업자번호 중복방지, 크레탑 분리 분석, 정책자금 매칭, 회원관리 기능 유지
- v3.6.1 안정 패키지 버전 유지
