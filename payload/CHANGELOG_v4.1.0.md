# CHANGELOG v4.1.0

## 등록 고객 기반 정책자금 자동매칭

### 신규 파일
- registered_policy_match.py

### 수정 파일
- app.py
- VERSION.txt

### 기능
- 정책자금 매칭 화면에서 기존 등록 고객 선택
- 크레탑 자동등록으로 생성된 누적 고객DB를 자동으로 읽음
- 선택 고객의 기업정보 미리보기
- 누락된 핵심 매칭정보 안내
- 선택 고객 1개만 포함된 임시 매칭파일 자동 생성
- 기존 main.py 정책자금 매칭 엔진 그대로 사용
- 매칭 결과와 실행이력 저장
- 기존 엑셀 업로드 매칭 기능 유지

### 데이터 보호
- 기존 누적 고객DB 및 고객리스트를 수정하지 않음
- 원본을 복사한 임시 파일로만 매칭 실행
- 상시정책자금DB·고용지원금DB 등 기존 시트와 서식 유지
