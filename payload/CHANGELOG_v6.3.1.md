# CHANGELOG v6.3.1

## 수정 사항
- 녹음파일 AI 캐시를 로컬 파일과 Supabase에 이중 저장하도록 개선했습니다.
- 업데이트·재배포 후 로컬 캐시가 없어도 Supabase에서 기존 녹취록과 상담일지를 복원합니다.
- 캐시 확인 순서를 `로컬 캐시 → Supabase 캐시 → 신규 AI 분석`으로 변경했습니다.
- 클라우드 캐시를 찾으면 로컬 캐시도 자동 복원하여 다음 조회 속도를 높였습니다.
- Supabase 연결 또는 캐시 테이블 오류가 발생해도 신규 상담 분석 기능은 계속 동작하도록 예외처리했습니다.

## 데이터 보호
- 기존 고객DB, CRM, 상담일지, 녹음파일, 매칭키워드, 사용자 계정은 변경하거나 삭제하지 않습니다.
- 기존 로컬 캐시는 그대로 유지합니다.
- 신규 Supabase 캐시 테이블만 추가합니다.
