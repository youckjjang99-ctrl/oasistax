# CHANGELOG v3.8.2

## 법인 등기자료 자동입력

### 신규 파일
- `registry_worker.py`
- `registry_runner.py`

### 주가평가 기능
- 법인 등기사항증명서 PDF 업로드
- 법인명, 법인등록번호, 본점소재지, 설립일 자동 추출
- 자본금, 발행할 주식 총수, 발행주식 총수, 1주의 금액 자동 추출
- 보통주·우선주 등 종류주식 정보 추출 시도
- 기존 고객DB·크레탑 값과 등기자료 비교표 표시
- 모든 자동 입력값 직접 수정 가능
- 자본금과 `발행주식총수 × 액면가액` 일치 여부 자동 검증
- 등기자료는 회원별 `registry_cache.json`에 별도 저장

### 데이터 보호
- 기존 고객DB 및 고객리스트는 수정하지 않음
- 주가평가 전용 저장파일만 사용

### 한계
- 이미지 스캔 PDF는 텍스트 추출이 안 될 수 있음
- 주주별 보유주식수·지분율은 등기사항증명서에 없을 수 있어
  주주명부 또는 주식등변동상황명세서가 별도로 필요
