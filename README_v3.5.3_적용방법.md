# v3.5.3 적용방법

1. 압축 안의 모든 내용을 기존 `정책자금자동화` 폴더에 풉니다.
2. `RUN_OASIS_V353_UPDATE.cmd`를 더블클릭합니다.
3. `[SUCCESS]`가 나오면 다음 명령으로 실행합니다.

```powershell
streamlit run app.py
```
