# 정책자금자동화 v6.3.2 적용방법

## 실행 방법
1. 이 ZIP을 기존 정책자금자동화 프로젝트 폴더에 압축 해제합니다.
2. `RUN_V6.3.2_UPDATE.cmd`를 더블클릭합니다.
3. 화면에 `UPDATE COMPLETE`가 표시되면 적용 완료입니다.
4. GitHub 저장소라면 add, commit, push도 자동으로 시도합니다.
5. Supabase에서 `supabase_v632_upgrade.sql`을 한 번 실행합니다.

## 이번 통합패치에 포함된 기능
- CRM의 `녹음파일 상담일지 보기`
- 페이지 이동·재접속 후 저장 상담일지 조회
- 상담일지 정책자금 키워드 자동반영
- 정책자금매칭 메뉴 최신 키워드 반영
- Supabase 상담일지 영구 저장
- Supabase AI 캐시 저장 및 동일 녹음파일 중복분석 방지
- 업데이트 실행파일 한글 깨짐 방지

## 데이터 보호
기존 고객DB, CRM, 상담일지, 기업 히스토리, 정책자금 키워드, Supabase 데이터 및 녹음파일은 삭제하지 않습니다.
