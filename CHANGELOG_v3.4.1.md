# CHANGELOG v3.4.1

## 원클릭 업데이트 실행 안정화

### 수정 사항
1. Windows에서 `python` 명령이 인식되지 않는 환경을 보완했습니다.
2. 업데이트 실행 시 다음 순서로 Python을 자동 탐색합니다.
   - `py -3`
   - `python`
   - 사용자 로컬 Python 설치 폴더
   - Program Files의 Python 설치 폴더
3. 한글 깨짐을 줄이기 위해 실행창 안내문은 영문·숫자 중심으로 표시합니다.
4. 기존 v3.4.0의 자동 백업, 복원, 시스템 관리 기능은 그대로 유지합니다.
5. 적용 버전을 `v3.4.1`로 변경했습니다.

### 기존 기능 영향
- 고객관리, 고객DB 매칭, 크레탑 자동등록, 실행이력, 회원승인 기능은 변경하지 않습니다.
- 업데이트 실행 방식만 안정화했습니다.
