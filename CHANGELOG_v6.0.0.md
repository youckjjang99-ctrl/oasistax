# CHANGELOG v6.0.0

## 다중소스 증거기반 정책자금 매칭

### 신규 공고소스
- 기존 기업마당 동기화 DB
- 기존 상시형 정책자금 DB
- 기존 고용지원금 DB
- K-Startup OpenAPI 연동 지원
- 중소벤처기업진흥공단 OpenAPI 연동 지원

### 중복 통합
- 공고명과 기관명을 기준으로 중복 공고 자동 병합
- 여러 소스에서 확인된 공고는 소스 목록 표시
- 더 상세한 지원대상·지역·기간 정보 우선 유지

### 증거기반 매칭점수
- 고객 매칭키워드
- 상담녹취에서 자동 반영된 키워드
- 관심지원분야
- 자금사용목적
- 업종
- 사업장 소재지
- 설립연도·창업업력
- 제외키워드
- 신청기간 종료 여부

### 오탐 방지
- 지역 불일치 감점
- 예비창업·업력 3년·7년 요건 감점
- 제외키워드 포함 공고 감점
- 종료 공고 강한 감점
- 추천 근거와 감점사유 동시 표시

### 비용
- OpenAI API를 사용하지 않는 규칙기반 점수계산
- 외부 공고 API 호출 외 추가 AI 비용 없음

### 데이터 보호
- 기존 고객DB 수정 없음
- 기존 정책자금 매칭엔진 유지
- 결과는 회원별 multi_source_policy_cache.json에 별도 저장
