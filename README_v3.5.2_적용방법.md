# v3.5.2 적용방법

1. 압축 안의 모든 내용을 기존 `정책자금자동화` 폴더에 풉니다.
2. `RUN_OASIS_V352_UPDATE.cmd`를 더블클릭합니다.
3. `[SUCCESS]`가 나오면 다음 명령으로 실행합니다.

```powershell
streamlit run app.py
```

정상 확인 후 GitHub에 반영합니다.

```powershell
git add .
git commit -m "v3.5.2 사업자번호 중복 고객 자동 차단"
git push
```

## 변경 결과

- 크레탑 자동등록: 같은 사업자번호는 저장 버튼이 비활성화됩니다.
- 고객DB 엑셀 업로드: 기존 누적DB에 같은 사업자번호가 있으면 해당 행을 저장하지 않습니다.
- 기존에 이미 쌓인 중복 고객은 자동 삭제하지 않습니다.
