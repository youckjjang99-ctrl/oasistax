# v5.5.2 적용방법

이번 패치는 녹음을 늦게 종료해 뒤쪽에 무음이 길게 포함된 경우,
빈 구간을 자동 제외하도록 수정합니다.

## 적용

1. 압축 안의 파일을 기존 정책자금자동화 폴더에 풉니다.
2. `RUN_V552_FIX.bat`를 실행합니다.
3. GitHub에 반영합니다.

```powershell
git add .
git commit -m "v5.5.2 상담녹음 무음구간 자동제외"
git push
```

4. Streamlit Cloud를 Reboot합니다.

## 변경 결과

예를 들어 전체 6개 구간 중 5번과 6번이 무음이라면:

```text
1~4번 구간: 정상 녹취
5~6번 구간: 자동 제외
상담일지: 정상 생성
```

녹음파일 전체가 무음인 경우에만 다음 안내가 표시됩니다.

```text
녹음파일 전체에서 인식 가능한 음성을 찾지 못했습니다.
```
