# CHANGELOG v6.3.2

## 수정 사항
- v6.3.0의 CRM `녹음파일 상담일지 보기` 메뉴와 정책자금 키워드 자동연동 기능을 다시 포함했습니다.
- v6.3.1의 Supabase AI 캐시 복원 및 중복분석 방지 기능을 함께 포함했습니다.
- 업데이트 실행파일의 한글 인코딩 문제를 제거했습니다.
- Windows CMD에서 안전하게 실행되도록 영문 전용 실행파일과 CRLF 줄바꿈을 적용했습니다.
- 업데이트 후 실제 루트 파일 적용 여부와 Python 문법을 자동 검증합니다.
- Git 저장소인 경우 변경 파일을 자동으로 add/commit/push 하도록 구성했습니다.

## 데이터 보호
- 기존 고객DB, CRM, 기업 히스토리, 상담일지, 로컬 AI 캐시, Supabase 데이터, Storage 녹음파일 및 사용자 계정은 삭제하거나 초기화하지 않습니다.
- 수정 대상 파일은 업데이트 전에 `_oasis_backups` 폴더에 자동 백업합니다.
