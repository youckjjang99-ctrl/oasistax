# OASIS CRM v9.8.2 적용 안내

## 적용 기준

- 현재 버전이 v9.8.0 또는 v9.8.1이면 적용할 수 있습니다.
- v9.8.1을 아직 적용하지 않았다면 v9.8.2만 적용해도 됩니다.
- Railway 환경변수와 Supabase SQL은 추가할 필요가 없습니다.

## 적용 방법

1. 압축파일을 기존 `정책자금자동화` 폴더에 그대로 풉니다.
2. `RUN_UPDATE_v9.8.2.bat`를 실행합니다.
3. `UPDATE_OK`, `MOBILE_PHONE_AUTO_EXCLUSION=DISABLED`, `PY_COMPILE=OK`, `RUNTIME_SMOKE_TEST=OK`를 확인합니다.
4. 동봉된 Git 명령어로 GitHub main에 반영합니다.

## 연락처 처리 기준

- 회사명·주소가 검증된 일반전화와 휴대전화를 모두 표시합니다.
- 휴대전화는 영업 연결 가능성이 높은 연락처로 포함하되 출처를 함께 표시합니다.
- 날짜, 사업자번호 조각 및 임의의 8자리 숫자는 전화번호로 저장하지 않습니다.

## 백업과 롤백

- 적용 전 파일은 `_update_backups/before_v9.8.2`에 자동 백업됩니다.
- 적용 또는 실행 테스트가 실패하면 자동 롤백됩니다.
