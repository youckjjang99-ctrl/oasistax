# OASIS CRM v9.4.1 적용방법

## 적용 순서

1. 현재 프로젝트 버전이 `v9.4.0`인지 확인합니다.
2. 패치 ZIP을 기존 `정책자금자동화` 폴더에 바로 압축 해제합니다.
3. 파일 덮어쓰기를 허용합니다.
4. `RUN_UPDATE_v9.4.1.bat`를 실행합니다.
5. `UPDATE_OK`, `RAILWAY_PDFPLUMBER=INSTALLED`, `RUNTIME_SMOKE_TEST=OK`를 확인합니다.
6. 동봉된 GitHub 명령어로 `main`에 업로드합니다.
7. Railway가 새 커밋을 배포하고 `pdfplumber` 설치를 완료할 때까지 기다립니다.
8. 배포 완료 후 브라우저에서 `Ctrl + F5`를 누르고 신고서를 다시 분석합니다.

## 주의

- Railway 재배포 전에는 기존 서버가 이전 의존성을 계속 사용할 수 있습니다.
- 잘못 표시된 `개인사업장 1`은 등록하지 않습니다.
- 이미 저장했다면 해당 잘못된 고객을 삭제한 후 다시 등록합니다.

## 백업과 롤백

- 적용 전 파일은 `_update_backups/v9.4.0_before_v9.4.1_날짜시간`에 자동 백업됩니다.
- 컴파일 또는 실행점검 실패 시 v9.4.0 파일로 자동 롤백합니다.
- `.git`과 사용자 데이터는 변경하지 않습니다.
