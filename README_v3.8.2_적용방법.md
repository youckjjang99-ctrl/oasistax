# v3.8.2 적용방법

1. 압축 안의 모든 파일을 기존 `정책자금자동화` 폴더에 풉니다.
2. `RUN_V382_UPDATE.bat`를 더블클릭합니다.
3. `[SUCCESS]`가 나오면 실행합니다.

```powershell
streamlit run app.py
```

## 사용방법

1. 주가평가 메뉴에서 고객을 선택합니다.
2. 필요하면 `기존 고객DB에서 재무정보 불러오기`를 누릅니다.
3. `법인 등기자료에서 주식정보 불러오기`를 엽니다.
4. 등기사항증명서 PDF를 업로드하고 `등기자료 분석`을 누릅니다.
5. 자본금·발행주식총수·1주의 금액을 확인하고 수정합니다.
6. 주가평가를 계산합니다.

기존 고객DB는 수정하지 않습니다.

```powershell
git add .
git commit -m "v3.8.2 법인 등기자료 자동입력"
git push
```
