# OASIS CRM v9.4.4 업데이트 안내

## 수정 목적

종합소득세 신고서에서 확인할 수 있는 개인사업자 기본정보·재무정보·세무정보를
기업정보 화면과 법인전환 분석에 최대한 반영합니다.

## 수정 파일

- `income_tax_return_parser.py`
- `utils.py`
- `enterprise_center.py`
- `system_precheck.py`
- `VERSION.txt`

## 주요 표시 항목

- 기본정보: 사업자유형, 사업장 소재지, 주업종코드, 기장의무, 신고유형,
  귀속연도, 과세기간
- 재무정보: 총수입금액, 필요경비, 영업손익, 각사업연도소득금액,
  자산총계, 부채총계, 자본총계
- 신고정보: 종합소득금액, 과세표준, 적용세율, 산출세액, 세액감면,
  세액공제, 결정세액, 납부·환급세액, 소득률, 필요경비율

## 적용 방법

1. 압축 내용을 기존 `정책자금자동화` 폴더에 그대로 덮어 풉니다.
2. `RUN_UPDATE_v9.4.4.bat`를 실행합니다.
3. `UPDATE_OK`, `PY_COMPILE=OK`, `RUNTIME_SMOKE_TEST=OK`를 확인합니다.
4. 기업등록에서 기존 종합소득세 신고서를 다시 분석합니다.
5. 해당 사업장을 선택하고 등록하면 기존 고객의 누락 항목이 갱신됩니다.

## 유의사항

- 설립일과 업종명은 이 신고서에서 직접 확인되지 않아 임의 생성하지 않습니다.
- 업종은 신고서에 기재된 주업종코드를 표시합니다.
- 각사업연도소득금액은 선택 사업장의 사업소득금액을 사용합니다.
- 신고서에 표준재무상태표나 표준손익계산서가 첨부된 경우에만
  자산·부채·자본·영업손익을 채웁니다.

## 자동 백업과 롤백

- 적용 전 기존 파일은
  `_update_backups/v9.4.3_before_v9.4.4_날짜시간`에 백업됩니다.
- 적용 또는 실행 점검 실패 시 기존 파일로 자동 복구됩니다.
