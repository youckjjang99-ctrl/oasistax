# OASIS CRM v9.8.8

## 변경 내용

- 저장된 영업후보 목록을 로그인 사용자별로 분리했습니다.
  - 본인이 저장한 업체만 목록·연락처·엑셀·메모 화면에 표시됩니다.
  - 다른 사용자가 저장한 업체 정보는 목록에서 보이지 않습니다.
- 전사 공통 중복제외는 유지·강화했습니다.
  - 다른 사용자가 이미 저장한 업체도 이후 검색 결과에서 제외됩니다.
  - 저장 직전에도 공통 중복제외를 한 번 더 실행해 담당 사용자 변경을 방지합니다.
- 메모 저장 시에도 로그인 사용자 소유 업체만 수정하도록 제한했습니다.

## 보존 사항

- DB발굴 지역·사업자유형·고용 증가·연락처 검색·검색 이력·설립연차,
  기존 영업후보와 Supabase 데이터는 삭제하지 않습니다.
