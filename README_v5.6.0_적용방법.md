# v5.6.0 적용방법

이번 업데이트는 상담 녹취에서 정책자금 관련 신호를 분석해
고객별 매칭키워드와 관심지원분야를 자동 보완합니다.

## 적용

1. 압축 안의 파일을 기존 정책자금자동화 폴더에 풉니다.
2. `RUN_V560_UPDATE.bat`를 실행합니다.
3. GitHub에 반영합니다.

```powershell
git add .
git commit -m "v5.6.0 상담녹취 정책자금 키워드 자동반영"
git push
```

4. Streamlit Cloud를 Reboot합니다.

## 사용

```text
기업 컨설팅
→ 고객 선택
→ CRM
→ 녹음파일 상담일지 생성
```

상담일지 초안에 다음 항목이 표시됩니다.

- 정책자금 관련성
- AI 신뢰도
- 추천 매칭키워드
- 추천 관심분야
- 녹취 판단근거
- 추가 확인질문

`상담일지 저장 시 정책자금 추천키워드를 자동 반영`을 체크한 뒤
상담일지를 저장하면 고객별 정책자금 매칭설정에 추가됩니다.

## 보호 원칙

- 기존 키워드는 삭제하지 않습니다.
- 새 키워드만 중복 없이 추가합니다.
- 신뢰도 60점 미만은 자동 반영하지 않습니다.
- 이익소각·가지급금·가업승계 등 정책자금과 무관한 상담만 있었다면
  키워드를 억지로 추가하지 않습니다.
