# v6.2.3 적용방법

이번 패치는 녹음파일이 화면에 표시됐는데도
`녹취 및 상담일지 생성` 버튼이 활성화되지 않는 문제를 수정합니다.

## 적용

1. 압축을 기존 정책자금자동화 폴더에 풉니다.
2. `RUN_V623_FIX.bat`를 실행합니다.
3. GitHub에 반영합니다.

```powershell
git add .
git commit -m "v6.2.3 녹음파일 버튼상태 자동복구"
git push
```

4. Streamlit Cloud를 Reboot합니다.

## 정상 화면

파일 업로드가 완료되면 다음 안내가 표시됩니다.

```text
업로드 완료 · 78.1MB
```

그 아래 파란색 `녹취 및 상담일지 생성` 버튼이 활성화됩니다.

이전 작업상태 때문에 버튼이 비활성화된 경우에는
`상태 초기화` 버튼을 누르면 됩니다.
