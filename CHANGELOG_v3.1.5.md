# v3.1.5 Change Log

## 변경 목적
- v3.1.4에서 상단 헤더 로고/문구 크기 변경이 화면에 체감되지 않던 문제 보정
- 로그인 화면 오아시스 로고를 과하지 않은 수준으로 확대

## 수정 파일
- app.py
- auth.py
- ui.py
- VERSION.txt
- CHANGELOG_v3.1.5.md

## 변경 내용
- 상단 헤더 로고 영역에 `oasis-topbar-logo` 클래스를 추가하고, 실제 이미지 폭을 365px 기준으로 적용
- 상단 헤더 제목/설명 글자 크기와 간격 보정
- 로그인 화면 로고 호출 크기를 260px → 320px로 변경
- 기존 사이드바 메뉴/동그라미 표시 방식은 유지

## 기존 기능 영향
- 기능 로직 변경 없음
- UI 표시 크기만 조정
