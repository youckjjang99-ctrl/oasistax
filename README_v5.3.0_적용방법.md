# v5.3.0 적용방법

이번 버전은 녹음 상담일지의 노이즈 보정과 법인컨설팅 주제 확장 업데이트입니다.

## 적용

1. 압축 안의 모든 파일을 기존 정책자금자동화 폴더에 풉니다.
2. `RUN_V530_UPDATE.bat`를 더블클릭합니다.
3. GitHub에 반영합니다.

```powershell
git add .
git commit -m "v5.3.0 노이즈보정 종합컨설팅 상담분석"
git push
```

4. Streamlit Cloud를 Reboot합니다.

## 사용

```text
기업 컨설팅
→ 고객 선택
→ CRM
→ 녹음파일 상담일지 자동작성
```

녹음 업로드 전에 선택할 수 있습니다.

- 카페·배경소음 감소 전처리: 기본 사용
- 강한 노이즈 제거: 커피머신·주변 대화가 큰 경우

## 자동 분석 주제

정책자금뿐 아니라 다음 분야를 함께 분석합니다.

- 이익소각
- 자기주식
- 가지급금·가수금
- 가업승계
- 상속·증여
- 정관개정
- 임원퇴직금
- 유족보상금
- 비상장주식평가
- 법인보험
- 세액공제
- 경정청구
- 고용지원금
- 기업인증·R&D
- 노무·재무구조 개선

## 결과

상담일지에 다음이 추가됩니다.

- 자동 분류된 상담주제
- 주제별 신뢰도
- 녹취 판단근거
- 추가 검토 방향
- 고객 제출 약속
- 상담자 후속업무
- 미해결 질문
- 고객 후속 카카오톡 문안
