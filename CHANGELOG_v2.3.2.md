# 정책자금자동화 v2.3.2 Change Log

## 변경 목적
누적 고객DB 다운로드 파일이 기존 고객DB 양식의 색상, 서식, 시트 구성을 유지하지 못하는 문제를 개선했습니다.

## 수정 파일
- utils.py
- VERSION.txt
- CHANGELOG_v2.3.2.md

## 변경 내용
1. 누적 고객DB 저장 방식을 pandas 새 파일 생성 방식에서 기존 고객DB 템플릿 복사/기입 방식으로 변경했습니다.
2. 고객DB 시트의 색상, 테두리, 열 너비, 행 높이 등 기존 서식을 유지하도록 수정했습니다.
3. 상시정책자금DB, 고용지원금DB, 코드표, 사용가이드 등 기존 시트를 삭제하지 않고 유지하도록 수정했습니다.
4. 기존 v2.3~v2.3.1에서 생성된 고객DB누적 파일도 읽어서 기존 양식으로 재저장되도록 마이그레이션 로직을 유지했습니다.
5. 관리용 컬럼(누적저장일시, 회원ID, 담당자명)은 고객DB 다운로드 파일에는 포함하지 않습니다.

## 기존 기능 영향 여부
- 기존 매칭 기능 삭제 없음
- 회원별 누적 고객DB 기능 유지
- 크레탑 PDF 자동등록 기능 유지
- 고객DB 엑셀 구조가 기존 양식과 동일해져 매칭 안정성이 개선됩니다.
