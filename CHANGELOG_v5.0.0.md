# CHANGELOG v5.0.0

## 고객관리 + 기업관리센터 통합

### 메뉴 통합
- 기존 `고객관리` 메뉴 제거
- 기존 `기업관리센터`를 `기업 컨설팅`으로 변경
- 중복된 CRM 화면을 하나로 통합

### 기업 컨설팅 통합 화면
- 기업 기본정보
- 재무정보
- CRM 상태·진행단계·중요도·담당자·예정일·상담메모
- 정책자금 매칭키워드·관심지원분야·제외키워드
- 자금사용목적·투자예정금액·투자예정시기
- 등기정보
- 주가평가 결과
- 기업 히스토리
- AI 빠른진단
- 대표 미팅 질문

### 호환성
- 이전 세션의 `고객관리` 메뉴값은 자동으로 통합 화면에 연결
- 기존 고객DB·CRM·정책자금·주가평가·기업 히스토리 데이터 유지
- 기존 기능은 삭제하지 않고 화면만 통합

### 데이터 보호
- 기존 고객DB 삭제·초기화 없음
- Supabase 및 로컬 CRM 자료 변경 없음
- 메뉴와 화면 연결만 정리
