# CHANGELOG v5.5.2

## 무음 구간 자동 제외

### 수정 배경
녹음을 종료하지 않아 뒤쪽에 긴 무음이 포함된 경우,
분할된 마지막 조각의 음성 변환 결과가 비어 있어 전체 작업이 실패했습니다.

### 개선 내용
- 음성이 없는 분할 조각은 자동 건너뛰기
- 무음 조각 하나 때문에 전체 상담일지가 실패하지 않도록 수정
- 실제 음성이 있는 조각만 순서대로 합쳐 녹취록 생성
- 원래 구간 번호를 유지해 녹취 순서 확인 가능
- 제외된 무음 구간 개수를 녹취록 하단에 표시
- 파일 전체가 무음일 때만 별도 오류 안내

### 유지 기능
- 25MB 초과 자동분할
- 카페 노이즈 전처리
- API 캐시 및 비용절감
- 진행상태 표시와 중복클릭 방지
- 기존 고객DB·CRM 변경 없음
