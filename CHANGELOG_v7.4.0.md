# CHANGELOG v7.4.0

## 직원현황

- 기업컨설팅 `AI 진단` 옆에 `직원현황` 탭 추가
- 4대보험 가입자명부 XLSX·XLS·CSV·PDF 업로드
- 엑셀의 여러 시트와 다중 머리글 자동 탐색
- 스캔 PDF는 기존 자동회전·한글 OCR 엔진 사용
- 성명·생년·자격취득일·자격상실일 자동 분석
- 직원명은 마스킹하여 표시·저장
- 주민등록번호 전체값은 저장하지 않음
- 현재 가입자·최근 취득자·청년·중장년·고령자·장기근속 요약
- 직원별 자격취득일·근속기간·가입상태 표시
- 기업별 이전 명부 업로드 이력 보존
- 기업히스토리에 직원명부 분석 기록 자동 추가
- Supabase `oasis_employee_rosters` 테이블 연동

## 고용지원금 매칭

- 정책자금 다중소스 매칭 시 최신 직원현황 자동 연동
- 현재 가입자 수를 고용지원금 추천 근거에 표시
- 최근 3개월·6개월 자격취득 인원을 신규채용 근거로 활용
- 청년·중장년·고령자 연령구간을 관련 지원금 점수에 반영
- 2년 이상 장기근속 인원을 계속고용·고용유지 판단에 반영
- 직원현황이 없는 기업은 기존 매칭방식 유지
- 채용일·실업기간·임금·근로계약·사업주 제외요건은 추가 확인사항으로 표시

## 비밀번호 변경

- 좌측 메뉴 하단에 비밀번호 변경 영역 추가
- 현재 비밀번호 검증
- 새 비밀번호 8자리 이상 적용
- 새 비밀번호 확인값 검증
- 변경 완료 시 다른 기기의 기존 로그인 세션 자동 만료
- 변경 후 로그인 화면으로 이동

## 데이터 보호

- 기존 고객DB·CRM·상담일지·정관·정책자금 결과 유지
- 기존 직원명부 버전 덮어쓰기 없음
- 직원 원문 주민등록번호 저장 없음
