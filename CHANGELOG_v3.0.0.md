# 정책자금자동화 v3.0.0 Change Log

## 기준 버전
- v2.3.6

## 변경 목적
- 기존 정책자금 자동매칭 시스템을 오아시스 내부 CRM + 정책자금 컨설팅 플랫폼 구조로 확장하기 위한 1단계 업데이트입니다.

## 수정 파일
- app.py
- VERSION.txt
- CHANGELOG_v3.0.0.md

## 주요 변경사항
1. 사이드바 메뉴 구조 도입
   - 기존 가로 메뉴 방식에서 왼쪽 사이드바 메뉴 방식으로 변경했습니다.
   - 메뉴: 홈, 고객관리, 고객DB 업로드/매칭, 크레탑 자동등록, 내 누적 고객DB, 실행이력, 담당자 통계, 회원 승인 관리.

2. 고객관리 기본 화면 추가
   - 회원별 누적 고객DB를 기준으로 고객 목록을 조회합니다.
   - 업체명, 대표자명, 사업자등록번호, 업종명 등으로 검색할 수 있습니다.
   - 업체 선택 시 기본정보, 재무정보, 인증/기술 현황, 메모/사업내용을 확인할 수 있습니다.

3. 내 누적 고객DB 메뉴 분리
   - 누적 고객DB 다운로드 기능을 별도 메뉴로 분리했습니다.
   - 기존 고객DB 양식과 시트 구조 유지 로직은 그대로 사용합니다.

4. 홈 화면 개편
   - v3.0 방향성에 맞게 CRM, 크레탑 자동등록, 정책자금 매칭 중심으로 안내 문구를 정리했습니다.

## 기존 기능 영향 여부
- 기존 회원가입/로그인/승인제 기능 유지.
- 기존 고객DB 업로드/매칭 기능 유지.
- 기존 크레탑 PDF 자동등록 기능 유지.
- 기존 실행이력/담당자 통계/회원 승인 관리 기능 유지.
- 기능 삭제 없음.

## 충돌 가능성
- app.py의 화면 구조가 변경되므로, 사용자는 기존 상단 메뉴가 아닌 왼쪽 사이드바 메뉴를 사용해야 합니다.
- 내부 기능 로직은 유지했으므로 매칭 결과 생성 방식에는 영향이 없습니다.
