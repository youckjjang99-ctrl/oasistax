# OASIS v8.5.3 안정화 패치 — 누적 이력 보존형

## 기능 수정

- Railway/Streamlit 환경의 한글 PDF 폰트 경로 오류 완화
- 크레탑 고객등록과 정책자금 키워드 저장 단계 분리
- 정책자금 키워드 정규화 및 안전 저장
- 주요 오류 자동 로그 기록
- 외부 라이브러리 Import 없이 문법검사만 수행

## 업데이트 엔진 수정

- 이전 패치폴더를 삭제하지 않음
- 이전 패치 ZIP을 삭제하지 않음
- 이전 BAT 및 업데이트 엔진을 삭제하지 않음
- 현재 패치폴더를 업데이트 후에도 보존
- 명시된 프로젝트 코드 파일만 백업 후 덮어쓰기
- 실패 시 이번 패치가 관리하는 대상 파일만 자동 복원
