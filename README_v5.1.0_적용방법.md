# v5.1.0 적용방법

## 주요 변경

- 기업 컨설팅 화면 디자인 개편
- CRM 상태·상담 진행단계·매출·당기순이익 카드 분리
- AI 진단 카드 디자인 개선
- AI 컨설팅 리포트를 AI 진단 탭에 통합
- 왼쪽 별도 `AI 컨설팅 리포트` 메뉴 제거
- AI 진단 탭에서 엑셀 리포트 다운로드

## 적용

1. 압축 안의 모든 파일을 기존 `정책자금자동화` 폴더에 풉니다.
2. `RUN_V510_UPDATE.bat`를 더블클릭합니다.
3. `[SUCCESS]`가 나오면 실행합니다.

```powershell
streamlit run app.py
```

## 확인

1. 왼쪽 메뉴에서 `AI 컨설팅 리포트`가 사라집니다.
2. `기업 컨설팅 → AI 진단`으로 이동합니다.
3. 상단 CRM 상태와 진행단계가 서로 다른 카드로 표시됩니다.
4. AI 진단 하단에서 엑셀 리포트를 내려받을 수 있습니다.
