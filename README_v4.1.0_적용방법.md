# v4.1.0 적용방법

## 수정 파일
- app.py
- VERSION.txt

## 신규 파일
- registered_policy_match.py

## 적용

1. 압축 안의 모든 파일을 기존 `정책자금자동화` 폴더에 풉니다.
2. `RUN_V410_UPDATE.bat`를 더블클릭합니다.
3. `[SUCCESS]`가 나오면 실행합니다.

```powershell
streamlit run app.py
```

## 사용

1. `정책자금 매칭` 메뉴를 엽니다.
2. 등록 고객 목록에서 업체를 선택합니다.
3. 고객정보를 확인합니다.
4. `선택 고객 정책자금 매칭 실행`을 누릅니다.
5. 결과를 확인하고 엑셀을 다운로드합니다.

기존 고객DB 파일 업로드 방식도 아래쪽에 유지됩니다.

## 데이터 보호

- 기존 누적 고객DB는 수정하지 않습니다.
- 선택 고객 1개가 포함된 임시 복사본으로 매칭합니다.
- 기존 지원사업 시트와 서식은 유지됩니다.
