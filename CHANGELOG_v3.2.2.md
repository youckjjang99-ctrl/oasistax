# v3.2.2 Change Log

## CRM 성능 최적화 및 실제 고객 수 계산 수정

### 수정 파일
- app.py
- VERSION.txt
- CHANGELOG_v3.2.2.md

### 변경 내용
1. 고객관리 화면에서 엑셀 양식의 빈 서식 행까지 고객으로 계산하던 문제를 수정했습니다.
2. `업체명` 또는 `사업자등록번호`가 실제로 입력된 행만 고객으로 인정합니다.
3. 고객 목록을 20개씩 페이지 단위로 표시하도록 변경했습니다.
4. 고객DB 엑셀 로딩 캐시 시간을 조정해 메뉴 이동 시 반복 로딩 부담을 줄였습니다.
5. 검색/필터 결과 건수를 별도로 표시하도록 개선했습니다.

### 기존 기능 영향
- 기존 고객DB 엑셀 파일 구조는 변경하지 않습니다.
- CRM 메모/상태/타임라인 저장 구조는 유지됩니다.
- 정책자금 매칭, 크레탑 자동등록, 회원관리 기능에는 영향이 없습니다.
