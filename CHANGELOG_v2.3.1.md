# CHANGELOG v2.3.1

## 목적
- 누적 고객DB 다운로드 파일을 기존 고객DB 양식과 동일하게 맞춤

## 변경 내용
- 누적 고객DB 저장 시 `누적저장일시`, `회원ID`, `담당자명` 컬럼을 제거
- 누적 고객DB 시트명을 `고객DB`로 저장
- 기존 고객DB 양식 컬럼 순서를 그대로 유지하도록 정규화 로직 추가
- v2.3에서 생성된 구버전 `고객DB누적` 시트도 읽어서 새 양식으로 변환 가능하게 처리
- 크레탑 PDF 자동등록 데이터도 기존 고객DB 컬럼에만 입력되도록 수정

## 기존 기능 영향
- 기존 매칭 기능 삭제 없음
- 고객DB 양식이 표준화되어 매칭 안정성 개선
- 이미 생성된 구버전 누적DB도 다음 저장 시 자동으로 표준 양식으로 정리됨
