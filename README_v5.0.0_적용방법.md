# v5.0.0 적용방법

## 변경 내용

기존의 두 메뉴:

```text
고객관리
기업관리센터
```

를 아래 하나의 메뉴로 통합합니다.

```text
기업 컨설팅
```

통합 화면에서 다음 기능을 모두 사용합니다.

- 기업정보
- 재무정보
- CRM
- 정책자금 매칭설정
- 등기정보
- 주가평가
- 기업 히스토리
- AI 진단
- 대표 미팅 질문

## 적용

1. 압축 안의 모든 파일을 기존 `정책자금자동화` 폴더에 풉니다.
2. `RUN_V500_UPDATE.bat`를 더블클릭합니다.
3. `[SUCCESS]`가 나오면 실행합니다.

```powershell
streamlit run app.py
```

## 확인

왼쪽 메뉴에서 기존 `고객관리`, `기업관리센터`가 사라지고
`기업 컨설팅` 하나만 표시되어야 합니다.

기존 고객DB와 CRM 자료는 그대로 유지됩니다.
