# 정책자금자동화 v3.2.1 Change Log

## 목적
- 사이드바 메뉴 클릭 및 화면 전환 시 느리게 보이는 현상을 완화합니다.
- 고객관리 화면에서 누적 고객DB 엑셀을 반복 로딩하지 않도록 최적화합니다.

## 수정 파일
- app.py
- ui.py
- VERSION.txt
- CHANGELOG_v3.2.1.md

## 변경 내용
1. 고객관리 누적DB 읽기 캐시 적용
   - 같은 파일은 메뉴 이동 때마다 다시 읽지 않고 캐시를 사용합니다.
   - 누적DB 파일이 변경되면 수정시간 기준으로 자동 갱신됩니다.

2. 메뉴 렌더링 구조 개선
   - 여러 메뉴 조건을 각각 검사하는 방식에서 선택 메뉴만 렌더링하는 `if / elif` 구조로 정리했습니다.
   - 기존 기능은 삭제하지 않았습니다.

3. Streamlit 재실행 페이드 체감 완화
   - 화면 전환 시 반투명하게 보이는 기본 효과를 CSS로 완화했습니다.

## 기존 기능 영향
- 회원가입/승인, 고객DB 매칭, 크레탑 자동등록, 누적DB 다운로드, CRM 기능은 유지됩니다.

## 배포
- GitHub에 수정 파일 업로드 후 Streamlit 재배포가 필요합니다.
