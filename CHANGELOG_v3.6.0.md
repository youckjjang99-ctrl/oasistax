# CHANGELOG v3.6.0

## 크레탑 PDF 분석 엔진 분리

### 수정 파일
- `app.py`
- `utils.py`
- `VERSION.txt`

### 신규 파일
- `cretop_worker.py`
- `cretop_runner.py`

### 주요 변경
1. PDF 분석을 Streamlit 앱과 별도 프로세스로 분리했습니다.
2. PDF 라이브러리가 segmentation fault로 종료돼도 웹앱 본체는 유지됩니다.
3. 고정 페이지 번호를 사용하지 않고 모든 페이지를 순차적으로 탐색합니다.
4. 사업자번호가 발견될 때까지 동적으로 탐색한 후 중복 여부를 먼저 확인합니다.
5. 중복 고객은 전체 재무분석을 진행하지 않습니다.
6. 신규 고객은 문서 전체에서 기업개요·재무제표·인증 섹션을 찾아 분석합니다.
7. 업로드 PDF를 메모리에 복제하지 않고 디스크로 스트리밍 저장합니다.
8. 분석 결과는 JSON으로 반환하며, 세션에는 작은 결과값만 보관합니다.
9. 분석 진행 로그와 비정상 종료 메시지를 화면에서 확인할 수 있습니다.

### 기존 기능 영향
- 고객관리, 사업자번호 중복차단, 정책자금 매칭, 회원관리 기능은 유지됩니다.
