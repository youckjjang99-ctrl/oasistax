# OASIS CRM v9.8.7

## 변경 내용

- 고객관리 기업 기본정보에 `설립연차`를 추가했습니다.
  - 설립일/설립년도를 기준으로 현재 연차를 계산합니다.
  - 3년차 이하는 `신규기업`, 4~7년차는 `성장초기`, 8년차 이상은
    `기존기업`으로 함께 표시합니다.
  - 설립일이 없거나 형식이 올바르지 않으면 `확인 필요`로 표시합니다.
- DB발굴 수집 지역을 17개 시·도로 확대했습니다.
  - 기존 서울특별시·경기도는 그대로 유지됩니다.
  - 광역시, 세종, 강원·전북 특별자치도, 제주를 포함한 전국 시·도를
    선택할 수 있습니다.
- 주소 기반 지역 정규화도 전국 시·도 기준으로 확장했습니다.

## 보존 사항

- 기존 고객DB, 설립일 입력·수정, 기업등록, DB발굴, 연락처 검색,
  고용 증가 필터, 검색 이력, 영업후보 저장 기능은 유지됩니다.
- 기존 Git 연결과 Supabase 구조·데이터는 변경하지 않습니다.
