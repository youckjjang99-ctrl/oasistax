# CHANGELOG v7.1.0

## 기업고객 검색·필터·휴지통 관리

- 기업컨설팅 상단에 실시간 고객 검색 추가
- 업체명, 대표자, 사업자번호, 전화번호, 지역, 업종, 담당자 통합검색
- CRM 상태, 상담 진행단계, 중요도, 담당자 필터 추가
- 오늘 조치, 기한 경과, 향후 7일, 7일 이상 미접촉 빠른 보기
- 중요도 4~5, 진행 중 고객, 계약완료 빠른 보기
- 활성 고객, 오늘 조치, 기한 경과, 휴지통 현황 카드 추가
- 검색결과를 기한 경과 → 오늘 조치 → 중요도 순으로 자동정렬
- 선택 고객 휴지통 이동 기능 추가
- 휴지통 이동 시 업체명 재입력과 삭제사유 입력 필수
- 원본 고객DB, CRM, 상담일지, 정책자금, 정관, 히스토리 보존
- 휴지통 고객 복원 기능 추가
- 삭제·복원 내역을 기업히스토리에 자동 기록
- Supabase `oasis_customer_trash` 테이블로 여러 기기 간 휴지통 상태 동기화
- 기존 기능과 데이터 구조 유지
