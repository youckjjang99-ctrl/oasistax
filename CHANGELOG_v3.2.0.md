# 정책자금자동화 v3.2.0 Change Log

## 버전명
v3.2.0 - 오아시스 CRM 1.0

## 수정/추가 파일
- app.py
- crm.py 신규 추가
- VERSION.txt
- CHANGELOG_v3.2.0.md

## 변경 내용
1. 고객관리 화면을 CRM 형태로 확장했습니다.
2. 누적 고객DB를 기준으로 고객 검색/상태 필터 기능을 추가했습니다.
3. 고객별 상태 관리 기능을 추가했습니다.
   - 신규
   - 상담중
   - 자료요청
   - 제안서 발송
   - 신청준비
   - 신청완료
   - 계약완료
   - 보류
4. 고객별 다음 액션과 다음 예정일 입력 기능을 추가했습니다.
5. 고객별 상담 메모 저장 기능을 추가했습니다.
6. 고객별 타임라인 저장/조회 기능을 추가했습니다.
7. CRM 데이터는 기존 고객DB 엑셀 파일을 변경하지 않고, 회원별 `user_data/{회원ID}/crm_data.json`에 별도 저장되도록 설계했습니다.

## 기존 기능 영향 여부
- 기존 고객DB 업로드/매칭 기능 유지
- 기존 크레탑 자동등록 기능 유지
- 기존 누적 고객DB 다운로드 기능 유지
- 기존 회원가입/승인 기능 유지
- 기존 고객DB 엑셀 양식 변경 없음

## 주의사항
- CRM 상담메모와 타임라인은 엑셀 고객DB가 아니라 별도 JSON 파일에 저장됩니다.
- GitHub에는 `user_data/`가 `.gitignore`로 제외되어 있어야 합니다.
- Streamlit Cloud에서는 저장공간이 영구 DB가 아니므로, 장기적으로는 SQLite 또는 외부 DB 연동을 권장합니다.
