# v4.1.2 적용방법

1. 압축 안의 모든 파일을 기존 `정책자금자동화` 폴더에 풉니다.
2. `RUN_V412_UPDATE.bat`를 더블클릭합니다.
3. `[SUCCESS]`가 나오면 실행합니다.

```powershell
streamlit run app.py
```

## 사용방법

### 크레탑 신규 고객
1. 크레탑 PDF 분석
2. 매칭키워드·관심분야·제외키워드 입력
3. 고객DB에 추가

### 기존 고객
1. 정책자금 매칭
2. 등록 고객 선택
3. 고객별 매칭설정 입력 또는 수정
4. 정책자금 자동매칭 실행

매칭설정은 별도 JSON에 저장되며 기존 고객DB는 수정하지 않습니다.

```powershell
git add .
git commit -m "v4.1.2 고객별 정책자금 매칭키워드"
git push
```
