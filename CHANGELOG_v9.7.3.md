# OASIS CRM v9.7.3 변경 내역

## Windows 실행 안정화

- `RUN_UPDATE_v9.7.3.bat`을 영문 ASCII 명령만 사용하도록 변경했습니다.
- UTF-8 한글이 Windows 명령 프롬프트에서 배치 구문을 분리하는 문제를 제거했습니다.
- `if errorlevel` 구문 대신 종료코드를 변수에 저장해 비교합니다.

## 포함 기능

- v9.7.2의 주식회사 전용 수집, 농업회사법인·유한회사 제외, KIPRIS 비활성화 및 연락처 보강 기능을 함께 적용합니다.
- 기존 고객·영업후보·Supabase 구조와 데이터는 변경하거나 삭제하지 않습니다.
