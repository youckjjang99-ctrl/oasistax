# OASIS CRM v9.4.0 적용방법

## 적용

1. 현재 프로젝트 버전이 `v9.3.4`인지 확인합니다.
2. 패치 ZIP을 기존 `정책자금자동화` 폴더에 바로 압축 해제합니다.
3. 별도 폴더 안이 아니라 프로젝트 루트에 `payload`, 적용기, 실행파일이 보이는지 확인합니다.
4. `RUN_UPDATE_v9.4.0.bat`를 실행합니다.
5. 화면에 `UPDATE_OK`, `PY_COMPILE=OK`, `RUNTIME_SMOKE_TEST=OK`가 표시되는지 확인합니다.
6. 앱을 실행해 좌측 `기업등록` 메뉴를 확인합니다.

## 사용

- 법인사업자는 `법인사업자 - 크레탑 PDF`를 선택합니다.
- 개인사업자는 `개인사업자 - 종합소득세 신고서`를 선택합니다.
- 신고서 분석 후 등록할 사업장을 선택합니다.
- 사업장별 법인전환 검토 결과를 확인한 뒤 선택 사업장을 등록합니다.

## 백업과 롤백

- 적용 전 파일은 `_update_backups/v9.3.4_before_v9.4.0_날짜시간`에 자동 백업됩니다.
- 컴파일 또는 실행점검이 실패하면 기존 파일로 자동 롤백합니다.
- `.git`, 기존 사용자 자료, 기존 백업 폴더는 변경하지 않습니다.

## GitHub

업데이트 확인 후 `GITHUB_UPLOAD_COMMANDS_v9.4.0.txt`의 명령어를 PowerShell에서 순서대로 실행합니다.
