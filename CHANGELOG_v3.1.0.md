# 정책자금자동화 v3.1.0 Change Log

## 목적
- v3.0.0에서 도입한 사이드바 기반 메뉴 구조를 오아시스 내부 업무 시스템에 맞게 시각적으로 개선했습니다.

## 수정 파일
- app.py
- VERSION.txt
- CHANGELOG_v3.1.0.md

## 주요 변경사항
1. 좌측 사이드바 디자인 리뉴얼
   - 오아시스 로고 크기 확대
   - 브랜드명 및 시스템 설명 추가
   - 로그인 사용자 카드 디자인 추가
   - 메뉴 글자 크기 확대
   - 메뉴별 아이콘 추가
   - 선택된 메뉴 강조 스타일 적용

2. 전체 UI 톤 개선
   - 오아시스 블루 계열 배경 적용
   - 카드형 화면 구성 강화
   - 버튼, 다운로드 버튼, 파일 업로드 영역 디자인 개선
   - 상단 헤더를 업무 시스템 형태로 개선

3. 기존 기능 유지
   - 회원가입/로그인/승인 기능 유지
   - 고객관리 기능 유지
   - 고객DB 업로드/매칭 기능 유지
   - 크레탑 자동등록 기능 유지
   - 내 누적 고객DB 다운로드 기능 유지
   - 실행이력/담당자 통계/회원관리 기능 유지

## 기존 기능 영향 여부
- 기능 로직 변경은 최소화했습니다.
- 주로 CSS와 사이드바 메뉴 표시 방식만 개선했습니다.
- 기존 데이터 저장 방식 및 매칭 로직에는 영향을 주지 않습니다.

## 재배포
- GitHub 업로드 후 Streamlit 재배포가 필요합니다.
