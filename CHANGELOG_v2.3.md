# 정책자금자동화 v2.3 Change Log

## 추가 기능
- 크레탑 기업종합보고서 PDF 업로드 기능 추가
- PDF에서 업체명, 대표자명, 사업자등록번호, 법인번호, 주소, 업종, 종업원수, 설립일, 재무정보 자동 추출
- 벤처, 이노비즈, 메인비즈, 연구개발전담부서, 기업부설연구소, 특허, 상표 여부를 Y/N 값으로 자동 변환
- 로그인 회원별 `고객DB누적.xlsx`에 크레탑 추출값을 1행으로 자동 추가
- 사업자등록번호 중복 시 기존 데이터 업데이트 / 저장하지 않기 / 새 행 추가 선택 기능 추가
- 추출값 저장 전 미리보기 표 제공

## 주의사항
- 텍스트 추출 가능한 크레탑 PDF 기준으로 동작
- 스캔 이미지형 PDF는 텍스트 추출이 제한될 수 있음
- 자동 추출 불가 항목은 공란 유지
