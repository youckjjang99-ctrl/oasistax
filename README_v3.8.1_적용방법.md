# v3.8.1 적용방법

1. 압축 안의 모든 파일을 기존 `정책자금자동화` 폴더에 풉니다.
2. `RUN_V381_UPDATE.bat`를 더블클릭합니다.
3. `[SUCCESS]`가 나오면 실행합니다.

```powershell
streamlit run app.py
```

## 해피월드 테스트

1. 주가평가 메뉴에서 해피월드를 선택합니다.
2. `기존 고객DB에서 재무정보 불러오기`를 누릅니다.
3. 과거 3개년 정보가 없으면 해피월드 크레탑 PDF를 한 번 분석합니다.
4. 다시 고객DB 불러오기를 누르면 저장된 재무 캐시가 반영됩니다.

기존 고객DB 파일은 수정하지 않습니다.

```powershell
git add .
git commit -m "v3.8.1 주가평가 오류 및 고객DB 연동 수정"
git push
```
